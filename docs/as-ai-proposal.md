# AI 기반 충전기 AS 코파일럿 제안서

## 1. 제안 배경
귀사는 다음 운영 이슈를 상시적으로 경험하고 있습니다.
- 24시간 콜센터 운영 부담
- 오접수 및 불필요 출동
- 진단 미숙으로 인한 재출동
- 보증 AS 장기화로 인한 비용 증가

본 제안은 위 문제를 **RAG 기반 AS 코파일럿**으로 해결해 유지보수 비용과 복구 시간을 동시에 줄이는 것을 목표로 합니다.

## 2. 제안 솔루션
### 2.1 서비스 개요
- 에러코드/증상 입력 시 유사 AS 사례를 검색
- 원인 후보, 조치 단계, 필요 부품을 제시
- 근거 출처를 함께 제공해 신뢰성 확보

### 2.2 확장 서비스 (추가)
- **AS 자재 판매 연계**: 진단 결과 기반 권장 자재 자동 제안/주문
- **지역별 AS 전문가 풀 연결**: 긴급 건에 권역별 전문 인력 자동 매칭
- **AI 기술콜센터(기술지원센터)**: 24/7 1차 기술응대, 접수 품질 표준화
- **충전소 설비 점검관리 SaaS**: 사이트 단위 점검 계획, 이력, 리포트 관리

### 2.3 도입 효과
- 오접수 감소 및 출동 필터링
- 재출동률 감소로 인력 운영 효율화
- FTFR 향상으로 고객 만족도 개선
- 외주 인력 숙련도 편차 완화
- 부품 미지참 출동 감소로 현장 생산성 개선
- 콜센터 1차 해결률 향상 및 야간 대응 안정화

## 3. 구축 범위
- AS 이력/매뉴얼/장애코드 데이터 연계
- RAG 인덱스 구축 및 주기적 업데이트
- 콜센터/현장용 조회 UI
- KPI 대시보드(오접수, 재출동, FTFR, MTTR)
- 자재 카탈로그/재고 연동(옵션)
- 전문가 풀 매칭 워크플로우(옵션)
- 점검관리 SaaS(일정, 점검표, 점검이력) 모듈(옵션)

## 4. 추진 일정 (예시: 12주)
- 1~2주: 요구사항 확정, 데이터 진단
- 3~6주: 데이터 정제 및 RAG 구축
- 7~9주: 파일럿 운영(현장/콜센터)
- 10~12주: KPI 검증 및 상용 전환 제안

## 5. 성과 측정 기준
- 오접수율
- 재출동률
- FTFR
- MTTR
- 월간 출동 원가 절감액
- 콜센터 1차 해결률
- 부품 미지참 재방문율
- 전문가 매칭 소요시간

## 6. 운영/보안 정책
- AI는 권고안 제공, 최종 판단은 엔지니어 수행
- 고객 데이터는 계약 범위 내에서만 사용
- Private Cloud/On-Premise 옵션 제공 가능

## 7. 상용화 모델 (요약)
- 초기 구축비 + 월 구독료
- 관제/API 연동 시 사용량 과금
- 성과연동형 요금제 옵션
- AS 자재 판매 마진 모델
- 전문가 매칭 건당 수수료 모델
- AI 기술콜센터 위탁 운영 모델

## 8. 요청 사항
- 파일럿 대상 사이트 지정(1~2곳)
- 최근 12~24개월 AS 이력 제공
- 현장 리더 및 콜센터 담당 각 1명 지정

## 9. 기대 ROI (계산식)
- 절감액 = (감소 출동건수 × 건당 출동비용) + (단축시간 × 시간당 인건비)
- 투자 회수 기간 = (초기 구축비 + 6개월 운영비) / 월 절감액

## 10. 결론
본 제안은 단순 AI 도입이 아닌, **운영 KPI를 개선하는 비용 절감 프로젝트**입니다.  
파일럿을 통해 정량 성과를 확인한 뒤 단계적으로 확대하는 방식이 가장 안전하고 빠른 상용화 경로입니다.
