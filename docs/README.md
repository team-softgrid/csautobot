# CSAutobot 문서 인덱스

> CSAutobot는 aiCSMS가 수집·분석한 데이터를 **설명하고 보고하는** AI 어시스턴트다.
> 데이터를 직접 수집하거나 CSMS에 접근하지 않는다.

## 핵심 원칙

```
CSAutobot은 "말하는 레이어"다.
데이터 수집은 aiCSMS, 분석도 aiCSMS,
CSAutobot은 그 결과를 사람이 이해할 수 있게 전달한다.
```
