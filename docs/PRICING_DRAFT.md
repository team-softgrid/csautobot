# 요금제 초안 (월 과금)

## 목표

- 초기 확산을 위한 무료 플랜 제공
- 운영사가 실제 비용 절감 효과를 체감하면 유료 전환
- 토큰/추론비를 통제하면서 마진이 남는 구조 설계

## 플랜 구성

## Free

- 대상: 소규모 PoC, 파일럿 현장
- 포함:
  - 월 질의 제한
  - 기본 유사사례 검색
  - 단일 사업장/제한된 사용자
- 제한:
  - 고급 분석 리포트 없음
  - API 연동 없음

## Pro

- 대상: 실운영 현장
- 포함:
  - 점검일지 AI 어시스턴트
  - 사진 첨부 기반 요약
  - 월간 분석 리포트
  - 다수 사용자/사업장 지원(상한 포함)
- 과금 기준(권장):
  - 기본 구독료 + 추가 AI 사용량(질의 수 또는 토큰)

## Enterprise

- 대상: 대형 운영사/프랜차이즈
- 포함:
  - 멀티 테넌시/SSO/감사로그
  - 전용 SLA
  - 전용 배포(필요 시 VPC/온프렘)
  - 커스텀 모델/룰
- 과금 방식:
  - 연간 계약 + 사용자/설비 규모 기반

## 단가 산정 가이드

- 원가 요소:
  - LLM 추론비
  - 임베딩/인덱싱 비용
  - 스토리지/트래픽
  - 운영/지원 인건비
- 가격 설계 원칙:
  - Free는 전환 채널, 원가 최소화
  - Pro는 확실한 ROI(시간 절감, 재방문 감소) 기준으로 책정
  - Enterprise는 계약 기반 커스텀

## 권장 KPI

- Free → Pro 전환율
- 계정당 월 활성 사용자수(MAU)
- 유료 고객 유지율(리텐션)
- 고객당 평균매출(ARPA)
